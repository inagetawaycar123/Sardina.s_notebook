```c
extern int* BubbleSort(int* array, int length){
    int* head = array;
    int count = 0;
    do{
        count = 0;
        for(int i=0; i < length-1; i++){
            if(array[i] > array[i+1]){
                int temp = array[i];
                array[i] = array[i+1];
                array[i+1] = temp;
                count++;
            }
        }
    }while(count!=0);
    return head;
}
```

## 核心原理

基于 “相邻元素比较交换” 的贪心策略，具体逻辑：

1. 从数组起始位置开始，依次比较**相邻的两个元素**（索引 `i` 和 `i+1`）；
2. 若前一个元素大于后一个元素（升序排序场景），则交换两者位置；
3. 一轮遍历后，数组中**最大的元素**会被 “推到” 数组末尾（完成一次 “冒泡”）；

> 降序排序只需将 “前元素大于后元素” 改为 “前元素小于后元素” 即可。

**时间复杂度：$O(n^2)$**
