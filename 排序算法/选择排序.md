```c
extern int* SelectSort(int *array, int length){
    int* head = array;
    for(int i = 0; i < length - 1; i++){
        int temp = array[i];
        int pos = i;
        for(int j = i + 1; j < length; j++){
            if(temp > array[j]){
                temp = array[j];
                pos = j;
            }
        }
        array[pos] = array[i];
        array[i] = temp;
    }
    return head;
}
```

>在未排序的数组部分中找到最小的元素，并将其与第一个未排序的元素位置互换。

**时间复杂度：$O(n^2)$**
