# 极简HTML入门教程：
10分钟写个网页 HTML是网页的“骨架”，用标签描述文字、图片等内容，超简单！
1. 先搞懂核心结构（固定模板） 新建文本文件，改后缀为`.html`，复制下面内容，这是所有网页的基础框架： 
```html 
<!DOCTYPE html> <!-- 声明这是HTML5文件 --> 
<html> <!-- 网页根标签，所有内容都包在里面 --> 
<head> <!-- 网页“后台”，放标题、样式等（不直接显示） --> 
	<title>我的第一个网页</title> <!-- 浏览器标签上的标题 --> 
</head> 
<body> <!-- 网页“前台”，所有要显示的内容都放这 --> 
	这里写你想展示的内容！ 
</body> 
</html> ``` 
2. 必学基础标签（直接用） 
在`<body>`里加这些标签，立刻丰富内容： 
- 标题：`<h1>最大标题</h1>` 到 `<h6>最小标题</h6>`（1级最粗最大） 
- 段落：`<p>这是一段普通文字，自动换行，段落间有空隙</p>` 
- 图片：`<img src="图片路径" alt="图片加载失败时显示的文字">`（路径可以是网上图片链接，也可以是本地图片位置） 
- 链接：`<a href="跳转地址">点击我</a>`（href里填网址，点文字就跳转到对应页面）
 3. 实战：写个简单网页 把下面代码复制到`.html`文件，用浏览器打开，就能看到效果： 
```html
<!DOCTYPE html> 
<html> 
<head> 
	<title>我的小网页</title> 
</head> 
<body> 
	<h1>欢迎来到我的网页！</h1> 
	<p>这是我用10分钟学会HTML写的第一个页面~</p> 
	<img src="https://img0.baidu.com/it/u=3444469941,1443543899&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=300" alt="好看的风景图">
	<p>点这里去<span style="color:red">百度</span>：<a href="https://www.baidu.com">百度一下</a></p> 
</body> 
</html> ``` 
   4. 小技巧 
   - 标签大多成对出现（如`<p>`和`</p>`），少数单标签（如`<img>`） 
   - 想临时改样式，用`<span style="属性:值">`，比如上面代码里的红色文字
   - 写完保存后，双击文件就能用浏览器预览效果
